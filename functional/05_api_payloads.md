# Примеры больших payload из ключевых роутов

В этой папке содержатся примеры реальных payload, которые возвращают ключевые роуты бэкенда. Эти файлы используются для анализа структуры данных без необходимости запуска проекта.

---

## **Список файлов**

### 1. База знаний

- **Файл**: [`05_api_payloads/knowledge-base__payload.txt`](05_api_payloads/knowledge-base__payload.txt)
- **Описание**:
  Пример payload, возвращаемого роутом `GET /api/knowledge-base`.
  Содержит полную структуру базы знаний, включая:
  - Произведения (`works`) с отрывками (`excerpts`) и заданиями.
  - Поэтов (`poets`) с стихотворениями (`poems`) и заданиями.
  - Блоки заданий (`block3`).
  - Настройки (`settings`).
- **Объём**: ~9.5 МБ (JSON).
- **Использование**:
  - Анализ структуры данных для фронтенда.
  - Кеширование и инвалидация при обновлении базы через админку.

---

### 2. Прегенерированный вариант

- **Файл**: [`05_api_payloads/variants-runtime-pregenerated__payload.txt`](05_api_payloads/variants-runtime-pregenerated__payload.txt)
- **Описание**:
  Пример payload, возвращаемого роутом `GET /api/variants/runtime/pregenerated`.
  Содержит структуру сгенерированного варианта ЕГЭ, включая:
  - Задания части 1 (краткий ответ).
  - Задания части 2 (развернутый ответ).
  - Связи с базой знаний (авторы, произведения, отрывки, стихотворения).
- **Объём**: Зависит от сложности варианта.
- **Использование**:
  - Отображение прегенерированного варианта на странице `/new_test`.
  - Кеширование для ускорения загрузки.
